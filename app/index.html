Potree platfOrm for iNfrasTructure Inspection is a custom web-based platform that supports decision-makers in the process of digital documentation and management of bridges assets. In particular, it provides user-friendly tools for exploring interactively direct-from-survey 3D products coming from in-situ inspection with UAV, photogrammetry and laser scanning. It is entirely developed with free and open-source technologies.

The webmap of the welcome page is based on Bootleaf, a simple, responsive template for building web mapping applications with Bootstrap 3, Leaflet, and typeahead.js.

The 3D interactive viewer are based on Potree, Cesium and Three.js.

The data provided on this site is for informational and planning purposes only.

This project is currently under heavy development!
